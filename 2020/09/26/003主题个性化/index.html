给hexo换个好看的主题

2020-09-26

按：hexo博客默认的主题的完整度还是挺高的，但是美观度不佳，因此我想换一个好看的主题。而hexo的用户群也不小了而且有很多博主把自己的主题分享官网的主题库给大家使用。

一、挑选主题

官网|主题 下有很多主题展示，可以从中选择一个来使用，下载多个来试用也是可以的。

![微信图片_20200927103847](C:\Users\LIN\Desktop\blog pictures now\change_theme\微信图片_20200927103847.png)

PS：点击主题预览图可进入示例用博客，点击主题名称可进入主题GitHub仓库。

二、下载主题

以主题 Repica 为例，点击主题名称后会进入以下界面：

滚轮稍微往下啦，会看到一个可预览的REDME.md文件（不是所有主题都有的），这是一个对于该主题的一个介绍文档——可能会有主题简介、版本号、主题安装代码等信息。

1、Git命令行克隆（常用）

像 Repica 的readme里就直接有给出 git 命令行的代码，这点说明了它对新手还是挺友好的。

Install（安装）

这两个代码二选其一就行了：

在hexo博客文件夹下启动 Git Bash Here 输入或者复制粘贴选择的代码（建议手打，可以增加熟悉度；git命令行下 shift+Ins 粘贴），后回车即可克隆主题到本地，打开 hexo 文件夹下的 themes 就能看到新下载的主题了。

![微信图片_20200927110749](C:\Users\LIN\Desktop\blog pictures now\change_theme\微信图片_20200927110749.png)

注：

对于克隆代码，对应不同的主题，只需要改动

代码中的网址https://github.com/sabrinaluo/hexo-theme-replica替换为你想使用的主题所在的网址即可。（themes/ 后的主题名称也更改为你选择的主题的）

2、压缩包下载（部分主题支持）

例如像主题 Fluid ，它会有 release版本 的文件，就相当于主题文件夹的一个压缩包，下载后解压并拷贝到 hexo 文件夹下的 themes 文件夹就可以了。

注：最好重命名新主题的文件夹名（格式为最简主题名称即可，去掉版本号及前缀）。

三、启用新主题

用记事本打开博客主站的 _config.yml 文件，修改 theme 后的信息，改为新选择的主题名称（大小写和主题文件名一致）。

然后在博客文件夹下 鼠标右键-Git Bash here 打开 git 命令行，分别输入代码：

hexo clean 删除生成的文件和缓存。

hexo g 生成文件

hexo s 搭建本地服务器

在浏览器输入http://localhost:4000/即可查看新主题下的博客了。（**git** 命令行下 Ctrl+c 就可以关闭本地服务器了）

如果觉得主题合适，就可以输入命令 hexo d ，把新的静态文件部署到 Github 上了。
