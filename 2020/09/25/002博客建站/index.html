个人博客初建

2020-09-25

按：近期发现了Hexo+GitHub pages这个快速简洁的独立博客架构，非常符合我的风格。这是尝试以偏小白向做的一个建博过程的文字版笔记。

Hexo简介

Hexo是一款基于Node.js的静态博客框架，依赖少易于安装使用，可以方便的生成静态网页托管在GitHub和Coding上，也许是非常适合搭建独立博客的框架。（hexo官网）

一、前期准备

系统：Windows 10

1、git安装

下载地址：https://gitforwindows.org/

直接点击download按钮下载，然后直接安装就好了

安装完成后，在桌面点击鼠标右键，选项增加了Git GUI Here和Git Bash Here即表明安装成功了

PS：1、接下来的操作主要在 git 命令行中执行——鼠标右键-Git Bash here——后出现的界面

​ 2、在git命令行中输入 git --version 后按 Enter键 可以查看git版本号

2、node.js安装

下载地址：https://nodejs.org/en/

点击12.xx.x LTS按钮进行下载即可，以及安装全过程next就好了。

安装结束后进入git命令行（鼠标右键-Git bash Here），分别输入node -v和npm -v，安装成功的话会如下显示版本号。

3、本地运行hexo博客

最后用 hexo s 命令来启动安装好hexo（第一次启动）

Ps：后面编辑过后的博客启动前或者部署到GitHub前需要先后键入命令

hexo clean 删除生成的文件和缓存

hexo generate 生成静态文件

不要关闭命令行，接着直接访问网页 http://localhost:4000 ，显示如下页面，本地博客搭建成功。命令行窗口下 Ctrl+c 关闭本地博客。

三、部署到GitHub

什么是Github？

GitHub是全球最大的的同性交友网站，对于程序员来说。emm，我嘛，是个混子。然后程序员可以把他们的代码托管在网站上边，所以这里会有很多开源的软件项目等等，博客网站也是以代码形式储存的所以也能存放在这里，而且是免费的。所以选择了它。

关于注册GitHub账号网上教程挺多的，可以自行去搜索下，或者直接官网注册也行

**Github网址: **https://github.com/
